## Детальные сценарии

### Сценарий 1: Продажа iPhone (Электроника)

**Пользователь**: Хочет продать iPhone 13 Pro на Авито

**Шаги**:

1. **Выбор категории**
   - Кликает на карточку "Электроника" 📱
   - Подсветка активной категории (border + scale анимация)

2. **Заполнение данных**
   ```
   Название модели: iPhone 13 Pro 256GB
   Цена: 65000
   Состояние: Нормальное (есть следы)
   Память/Характеристики: Графит, 256ГБ, АКБ 87%, Face ID работает
   Комплект: Коробка, оригинальный кабель, чехол в подарок
   ```

3. **Загрузка фото (опционально)**
   - Нажимает на зону загрузки
   - Выбирает файл (< 4MB)
   - Видит превью с кнопкой удаления

4. **Выбор тона**
   - Выбирает "Вежливый" ☕
   - Или "Продающий" ⚡ для быстрой продажи

5. **Генерация**
   - Нажимает "Сгенерировать с AI"
   - Видит спиннер "Думаю..."
   - Получает готовое объявление:
   ```markdown
   **iPhone 13 Pro 256GB в отличном состоянии**

   Продаю свой верный помощник — iPhone 13 Pro 256GB в цвете Графит.
   Телефону 2 года, но бережно использовался.

   **Характеристики:**
   - Память: 256GB
   - Аккумулятор: 87% здоровья
   - Face ID работает идеально
   - Не ремонтировался

   **В комплекте:**
   Коробка, оригинальный кабель Lightning, плюс чехол в подарок.

   Продажа связана с апгрейдом. Торг небольшой уместен.
   Пишите, отправлю дополнительные фото или видео!
   ```

6. **Умный совет**
   > 💡 "Сфотографируйте экран с включенным Face ID — это подтвердит работоспособность."

7. **Действия с результатом**
   - Копирует текст → Вставляет на Авито
   - Или нажимает "Прокачать SEO" → Получает ключевые слова

## Состояния интерфейса

### 1. Начальное состояние
- Все поля пустые
- Выбрана категория "Электроника" по умолчанию
- Тон: "Вежливый"
- Квота: 0/15

### 2. Заполнение формы
- Валидация на лету (после ввода)
- Обязательные поля отмечены красным при попытке генерации
- Автосохранение в localStorage (debounced 500ms)

### 3. Генерация
- Кнопка блокируется
- Спиннер анимация
- Прогресс квоты увеличивается
- Автоскролл к результату

### 4. Результат
- Markdown-рендеринг
- Кнопка копирования с обратной связью
- SEO-блок (если оптимизировано)
- Умный совет от AI

### 5. Повторная генерация
- Если есть результат → показывается подтверждение
- "Текущий результат будет потерян. Продолжить?"

## Обработка ошибок

### Ошибки валидации
```
❌ Поле "Название модели" обязательно
❌ Поле "Память" обязательно
```
Визуально: красная обводка поля + текст ошибки

### Технические ошибки
```
❌ Превышен лимит запросов. Подождите 45 сек.
❌ Не удалось сгенерировать. Проверьте соединение.
```

### Квота исчерпана
- Кнопка генерации disabled
- Индикатор квоты красный
- Таймер обратного отсчёта

## Модальные окна

### 1. RulesModal (Правила и тарифы)
Открывается по клику на иконку Info в шапке

**Содержание**:
- Стоимость действий (1-2 кредита)
- Пакеты пополнения (Starter/Seller/Pro)
- Условия использования

### 2. ConfirmationModal
Показывается при попытке перегенерации с существующим результатом

### 3. ShareModal
Опции:
- Копировать текст
- Копировать ссылку на приложение

## Edge Cases

### 1. Пустой localStorage (новый пользователь)
- Загружаются дефолтные значения
- Electronics как категория по умолчанию

### 2. Переполнение localStorage
- При ошибке квоты storage
- Fallback: сохранение без изображений (base64 слишком большие)

### 3. Медленное соединение
- Таймаут не установлен (ждём ответа Gemini)
- Возможность отмены только перезагрузкой страницы

### 4. Невалидный API ключ
- Сообщение: "Проверьте API ключ"
- 401 ошибка от Gemini API

### 5. Браузер без поддержки
- ES2020+ требуется
- HTTPS обязателен (для камеры/микрофона если добавится)
